Download & Install

Grab a prebuilt binary or install with Go.

Prebuilt Binaries

macOS (Apple Silicon) Download arm64 macOS (Intel) Download amd64
Linux (ARM64) Download arm64 Linux (x86_64) Download amd64

Binaries include checksums. Checksums (SHA256)

Install with Go

$ go install github.com/ywnaing/term@v0.1.0

Requires Go 1.24+. The term binary will be installed to $GOPATH/bin, usually ~/go/bin.

Verify

$ term --version
term 0.1.0

What term can do

Shortcuts

Define and run project shortcuts to automate frequent tasks.

term> /shortcut build
✓ go build ./...

Recipes

Save command recipes with notes and run them anytime.

term> /recipe db:migrate
✓ 3 commands

Explain

Get clear explanations for errors without leaving your terminal.

term> /explain error.txt
✓ Root cause explained

History

Browse and search your SQLite command history instantly.

term> /history sqlite
✓ 128 entries
